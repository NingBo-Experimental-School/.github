## 🚀 如何查看/下载照片？
1.  点击上方的 `Repositories` 查看所有活动文件夹。
2.  进入你想看的活动仓库。
3.  点击 `Releases` (发布版)，可以下载该次活动的所有照片打包文件。
4.  或者直接在仓库里浏览单张图片（点击图片文件，再点 `Download` 按钮）。


## ⚠️ 注意事项
-   照片仅用于校内宣传和纪念，未经许可请勿外传。
-   请尊重同学们的肖像权。
